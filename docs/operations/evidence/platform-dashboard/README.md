# Platform Dashboard Evidence

Store Vercel, Supabase, and related platform-control evidence that does not fit
another provider-specific folder.

Examples:

- Vercel deployment, alias, environment-variable, TLS, and health checks
- Supabase project health, migration, RLS, advisor, and Edge Function checks
- Cross-platform production readiness summaries

Do not store secrets, database row contents, customer data, or full environment
variable values.
